# Srinidhi Pattabiraman — Portfolio

Personal portfolio site targeting technical and GTM-adjacent roles: integrations engineering, technical support, and RevOps/customer-success engineering.

## Purpose

Built to make a strong impression on technical hiring managers at B2B SaaS companies in under 90 seconds. The site leads with quantified impact, shows a problem→solution→outcome arc on projects, and keeps the UI clean enough that the presentation itself signals craft.

## Stack

Plain HTML, CSS, and vanilla JS. No framework.

## Design principles

- Numbers beat adjectives
- Every section earns its place
- Polished over flashy — whitespace and legibility first
